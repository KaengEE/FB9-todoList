# React + Vite

Firebase 9버전을 활용한 ToDoList

사용자가 자신만의 TODO를 이용할 수 있도록 하였고

TODO의 수정 및 삭제가 가능합니다.

### 기능 구현

- firebase Authentication으로 로그인/회원가입
- firebase DB 활용
- 컬렉션 추가/삭제
- onSnapshot으로 실시간 처리
- 모달창으로 수정처리

### _추가기능(구현중...)_

- 오늘 날짜 표시
- 완료 체크
- 정렬 기능
- 검색
